Technical Terminology

Semantic HTML
Using meaningful elements like <article> and <nav> to describe page structure.
Accessibility
Designing content so it can be used by people with different needs, devices, and assistive technologies.
Responsive Design
Adapting layout and content so a page remains usable on phones, tablets, and desktops.
Version Control
Tracking and managing changes to project files using tools such as Git.
